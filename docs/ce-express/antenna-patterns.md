# Antenna Patterns

## Supported Formats
- `.msi`
- `.txt`

## Import Antenna Patterns
1. Open the **Antennas** tool in the CE Express toolbar
2. Click **Import**
3. Select your `.msi` or `.txt` pattern files
4. Patterns are saved to the antenna library

## Assign to a Cell
Open the cell in **Network Data Management** and set the antenna pattern field to the imported pattern name.
