# Workspace

## What is a Workspace?
A workspace is a project container in CE Express. It holds your network objects and links to a geodata folder (elevation, clutter rasters).

## Create a New Workspace
1. Click the **Workspace** tool in the CE Express toolbar
2. Click **Create New Workspace**
3. Enter a name
4. Set the geodata folder path (must contain `elevation.tif`)
5. Verify the coordinate system matches your raster files
6. Click **Save**

## Duplicate a Workspace
1. Open the **Workspace** tool
2. Hover over the workspace — three icons appear
3. Click the **Duplicate** icon
4. Enter a name and click **OK**

> **Tip:** If the new workspace doesn't appear immediately, refresh the page.
