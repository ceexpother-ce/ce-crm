# Network Objects

## What are Network Objects?
Network objects are the sites, cells, and repeaters you manage in CE Express — antennas, towers, microwave links.

## Add a New Object
1. Open **Network Data Management**
2. Click **Add New Record**
3. Fill in: `latitude`, `longitude`, `cell_name` (required)
4. Optional: height, azimuth, tilt, frequency, power, technology, antenna pattern
5. Click **Save**
6. Click **Synchronize Changes** to push to the database

## Import from CSV
1. Go to **Network Data Management**
2. Click **Import**
3. Select your CSV file (must have `latitude`, `longitude`, `cell_name` columns)
4. Map columns if needed and confirm
5. Click **Synchronize Changes**

## Move an Object
1. Select the object on the map
2. Click the **Move** tool
3. Left-click the new position — a yellow dot marks it
4. Click **Accept**

## Duplicate Objects
1. Select the objects to duplicate
2. Click **Duplicate selected objects**
3. Choose the target workspace
4. Click **Accept**
