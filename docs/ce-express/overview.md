# CE Express Overview

CE Express is a web-based radio network planning and optimisation tool built on ArcGIS Enterprise.

## What is CE Express?

CE Express allows users to manage network objects (sites, cells, repeaters), run RF predictions (RSRP, SINR, throughput), analyse coverage, run path profiles, and publish results to ArcGIS Portal — all from a browser.

## Key Features

- **Workspaces** — Organise projects with separate geodata and network objects
- **RF Prediction** — Calculate RSRP, SINR, throughput, and best-server coverage
- **Path Profile / Line of Sight** — Analyse terrain clearance between two points
- **Street View** — Split-screen map and street view navigation
- **Antenna Patterns** — Import and manage .msi / .txt antenna files
- **ArcGIS Portal Publishing** — Share prediction results as hosted feature layers
