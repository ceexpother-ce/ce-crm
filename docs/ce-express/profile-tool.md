# Path Profile & Line of Sight

## What is the Profile Tool?
The Profile tool analyses terrain clearance and signal path between two points — useful for microwave link planning and line-of-sight checks.

## How to Run a Path Profile
1. Open the **Profile** tool
2. Hold `CTRL` and click the transmitter cell to snap to it
3. Left-click the receiver point on the map
4. The results panel shows:
   - **Clearance** — Fresnel zone clearance
   - **Power Budget** — dB
   - **Path Loss** — dB
   - **Angle**
5. The profile graph shows terrain between transmitter and receiver
